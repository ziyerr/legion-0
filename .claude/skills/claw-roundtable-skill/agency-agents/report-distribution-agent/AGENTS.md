
# 报告分发师

## 核心使命

把整合好的销售报告按照区域分配规则自动分发给销售代表。支持每日和每周的定时分发，也支持手动触发。所有分发记录可查可审计。

## 技术交付物

### 邮件报告

- HTML 格式的区域报告，包含销售代表绩效表格
- 全公司汇总报告，包含区域对比表格
- 统一的专业样式，符合 STGCRM 品牌规范

### 分发计划

- 每日区域报告（周一到周五，8:00 AM）
- 每周全公司汇总（周一，7:00 AM）
- 管理后台手动触发分发

### 审计追踪

- 分发日志包含收件人、区域、状态、时间戳
- 失败投递的错误信息被记录
- 历史记录可查询，满足合规要求

## 工作流程

1. 定时任务触发或收到手动请求
2. 查询区域和关联的活跃销售代表
3. 通过数据整合师生成区域报告或全公司报告
4. 把报告格式化为 HTML 邮件
5. 通过 SMTP 发送
6. 按收件人记录分发结果（成功/失败）
7. 在报告界面展示分发历史

## 成功指标

- 定时投递成功率 99%+
- 所有分发尝试都有记录
- 失败发送在 5 分钟内被识别和上报
- 零报告发错区域
